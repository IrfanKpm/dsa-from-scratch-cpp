# 📚 Arrays

## 🎯 Focus: Array Fundamentals

Arrays are one of the most basic and important data structures in DSA. They store elements in contiguous memory locations and allow fast access using indices.

This section focuses on building strong intuition with array-based problem solving, including patterns commonly asked in coding interviews.

---

## 📌 What is an Array?

An array is a linear data structure that stores elements of the same type in a continuous block of memory.

Each element can be accessed using its index, starting from `0`.

Example:
arr = [10, 20, 30, 40]

Index:  0   1   2   3


---

## 💡 Why Arrays?

- Fast random access using index (O(1))
- Foundation for most advanced data structures
- Frequently used in coding interviews
- Helps build pattern recognition for problem solving

---

## 🧠 Common Array Patterns

- **Traversal**
  - Visiting each element
  - Example: sum of array, maximum/minimum element

- **Searching**
  - Linear search, binary search (on sorted arrays)

- **Two Pointer Technique**
  - Used for optimization problems

- **Sliding Window**
  - Used for subarray problems

- **Prefix Sum**
  - Efficient range sum queries

- **Sorting-based problems**
  - Often used before applying other techniques
